# Image Converter Pro — Alag-Alag Files

## Project Structure

```
image-converter-split/
│
├── index.html   ← Sirf HTML (structure)
├── style.css    ← Sirf CSS (design & layout)
├── script.js    ← Sirf JavaScript (logic)
└── README.md    ← Ye file
```

## Har File Kya Karti Hai

| File        | Kaam |
|-------------|------|
| `index.html` | Page ka structure — tabs, drop zones, buttons, panels |
| `style.css`  | Poora design — colors, fonts, animations, responsive layout |
| `script.js`  | Logic — file read karna, canvas conversion, DOCX banana, download |

## Kaise Chalayein

1. Teeno files ek hi folder mein rakho
2. `index.html` browser mein open karo
3. Done! ✅

> **Note:** Internet connection chahiye sirf pehli baar (Google Fonts + JSZip CDN load hoti hai).
> Baad mein kaam karta rahega.

## Conversions

- **JPG / JPEG → PNG** — Lossless, background color, scale
- **PNG → JPG** — Quality slider, background fill, scale  
- **JPEG → Word (.docx)** — Multiple images, title, layout options
